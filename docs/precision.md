**Precisions** exist for precisions when a variable is declared.
It's bascially like an asssertion but during the variable declaration !
the way it works is bascially like this: "variableName:type\[condition\]=value"
here we declared "variableName" of an unknown type that have a value that must fulfill the condition "condition"
**With Modifiers:**
it should be written like "variableName:modifier(type)\[condition\]=value"
